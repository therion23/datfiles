## Timex/Sinclair - 1000, 1500 and 2068
Origin: archive.org

The 1500 is functionally the same as a 1000 with a memory upgrade (a socalled RAM pack), so everything contained in that dat simply needs either model with enough memory.

These machines can be emulated in pure software, as well as on the ZX-Next and at least two other FPGA based solutions.

Thanks to https://www.timexsinclair.com/ for the great work.
